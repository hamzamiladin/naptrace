# Contributing to Naptrace

Thanks for your interest in contributing.

## Getting started

1. Fork and clone the repo.
2. Install Rust 1.83+ via [rustup](https://rustup.rs/).
3. Run `cargo build` to verify the workspace compiles.
4. Run `cargo test` to verify tests pass.
5. Run `naptrace doctor` to check your local environment.

## Making changes

- Create a branch from `main`.
- Keep commits focused — one logical change per commit.
- Run `cargo fmt` and `cargo clippy` before pushing.
- Add tests for new functionality.
- Update `CHANGELOG.md` under `[Unreleased]`.

## Pull requests

- Keep PRs small and reviewable.
- Describe what you changed and why.
- Link related issues.

## Code of conduct

Be respectful. We're all here to build useful security tooling.
